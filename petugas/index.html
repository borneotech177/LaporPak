Ketentuan Pengisian Laporan:
1. Isi Identitas Petugas dengan nama, shift, dan tanggal yang benar
2. Di Laporan Kegiatan & Kejadian: isi waktu dan uraian setiap kejadian/kegiatan selama shift. Foto bukti bisa diambil dari 📷 Kamera atau 🖼 Galeri
3. Di Inventaris Pos: centang semua barang yang ADA dan lengkap
4. Di Karyawan di Luar Jam Kerja: isi nama, jam datang, jam pulang, dan keterangan untuk setiap karyawan yang hadir di luar jam kerja
5. Di Serah Terima Shift: isi data petugas penyerah dan penerima, lalu ambil foto bukti serah terima. Pastikan area aman dan inventaris pos lengkap sebelum serah terima
6. Tanda tangani kotak TTD menggunakan jari langsung di layar
7. Klik Simpan untuk menyimpan PDF & Excel, atau Kirim WhatsApp untuk mengirim via WhatsApp

⚠ Semua field bertanda * wajib diisi sebelum menyimpan
✅ Pastikan foto serah terima diambil dengan jelas sebagai bukti

📋 KIRIM KE GOOGLE SPREADSHEETS
Cara setup (lakukan 1x):
1. Buka Google Sheets → Buat spreadsheet baru
2. Klik menu Extensions → Apps Script
3. Paste kode Apps Script di bawah → Deploy sebagai Web App
4. Copy URL Web App → paste di bawah
5. Data laporan akan otomatis masuk ke Sheet!
function doPost(e) { const ss = SpreadsheetApp.getActiveSpreadsheet(); let sh = ss.getSheetByName("Rekap") || ss.insertSheet("Rekap"); const d = JSON.parse(e.postData.contents); if (sh.getLastRow() === 0) { sh.appendRow(["Timestamp","Nama","Shift","Tanggal", "Kejadian","Kondisi","Inv Ada","Inv Tidak", "Karyawan","Penyerah","Shift Penyerah", "Penerima","Shift Penerima"]); } sh.appendRow([d.timestamp,d.namaPetugas,d.shift, d.tanggal,d.kejadian,d.kondisiLingkungan, d.inventarisAda,d.inventarisTidak, d.karyawanLuarJam,d.serahTerimaPenyerah, d.serahTerimaShiftPenyerah,d.serahTerimaPenerima, d.serahTerimaShiftPenerima]); return ContentService.createTextOutput("OK"); }
